# Snake Game 🐍

A basic snake game made using HTML, CSS, and JavaScript.

## Features

- Snake movement
- Food spawning
- Score counter
- Game over when snake hits wall or itself

## Files Used

- HTML
- CSS
- JavaScript

## How to Play

1. Download or clone the project
2. Open the HTML file in your browser
3. Use arrow keys to control the snake

## Controls

- ↑ Move Up
- ↓ Move Down
- ← Move Left
- → Move Right
